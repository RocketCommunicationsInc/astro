# Astro UXDS Lit Components

Welcome to the **Lit Components** version of Astro UXDS! 

This is a conversion of the Astro Web Components library from Stencil to Lit Element.

## What's Different?

- **Framework**: Built with [Lit](https://lit.dev/) instead of Stencil
- **Build Tool**: Uses Vite instead of Stencil's compiler
- **Same API**: Components maintain the same props, events, and slots as the Stencil versions
- **Same Styling**: Uses the same SCSS and design tokens from `@astrouxds/tokens`

## Getting Started

### Installation

```bash
npm install @astrouxds/astro-lit-components
```

### Usage

Import and use components in your application:

```html
<script type="module">
  import '@astrouxds/astro-lit-components/components/rux-button/rux-button.js'
</script>

<rux-button>Click Me</rux-button>
```

### With a Build Tool

```javascript
import '@astrouxds/astro-lit-components/components/rux-button/rux-button.js'
```

```html
<rux-button secondary>Secondary Button</rux-button>
```
