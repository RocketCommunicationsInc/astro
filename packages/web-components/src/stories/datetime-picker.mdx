# Datetime Picker

The Datetime Picker allows users to select specific dates from a calendar display for the purpose of minimizing data entry errors. Dates used are in [ISO 8601](https://www.iso.org/iso-8601-date-and-time-format.html) format and also include Julian ISO capability.
