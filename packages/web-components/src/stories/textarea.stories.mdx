## Customization

### Horizontal Labels

You can utilize the `form-field` and `label` Shadow Parts to have full control over label placement. For example:

```css
rux-textarea::part(form-field) {
    flex-direction: row;
}
rux-textarea::part(label) {
    margin-right: var(--spacing-input-label-left);
}
```

export const HorizontalLabel = (args) => {
    return html`
        <style>
            #left-example::part(form-field) {
                flex-direction: row;
            }
            #left-example::part(label) {
                margin-right: var(--spacing-input-label-left);
            }
        </style>
        <rux-textarea
            id="left-example"
            label="Label"
            type="text"
            placeholder="Text input"
        ></rux-textarea>
    `
}

    <Story name="Horizontal Label">{HorizontalLabel.bind()}</Story>
</Canvas>

## Cherry Picking

If you're already utilizing a build system that supports tree shaking and want to only import this individual component:

```js
import RuxTextarea from '@astrouxds/astro-web-components/dist/components/rux-textarea.js'
customElements.define('rux-textarea', RuxTextarea)
```
