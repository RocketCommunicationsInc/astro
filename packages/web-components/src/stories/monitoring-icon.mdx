# Monitoring Icon

These Icons represent objects, equipment, and concepts that are being administered or monitored. The purpose of these Icons is to easily, concisely, and clearly visually communicate their status to be to users.

## Cherry Picking

If you're already utilizing a build system that supports tree shaking and want to only import this individual component:

```js
import { RuxMonitoringIcon } from '@astrouxds/astro-web-components/dist/components/rux-monitoring-icon'
import { RuxStatus } from '@astrouxds/astro-web-components/dist/components/rux-status'

customElements.define('rux-monitoring-icon', RuxMonitoringIcon)
customElements.define('rux-status', RuxStatus)
```
