# Checkbox

A Checkbox describes a state or value that can be either "True or False.â€ Checkboxes are not mutually exclusive. More than one Checkbox may be checked at the same time.

For displaying invalid state, consider using a [Checkbox Group](?path=/docs/forms-checkbox-group--default-story) component.

### Indeterminate

An Indeterminate state (a dash symbol rather than checked) may display when a checkbox is used as a parent of a group of checkboxes where at least one child is selected and at least one is not.
