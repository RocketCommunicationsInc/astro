# Welcome to the Astro Storybook

This is the Astro UXDS Storybook, a sandbox for demonstrating and developing components and patterns built for
the Astro Space UX Design System ("Astro UXDS").

## Quick Start

For rapid prototyping, you can use our CDN to get up and running quickly.

Add the following to your `<head>` in any html file:

```
<link rel="preconnect" href="https://fonts.gstatic.com" />
<link
    href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;500;700&display=swap"
    rel="stylesheet"
/>
<link
    rel="stylesheet"
    href="https://unpkg.com/@astrouxds/astro-web-components/dist/astro-web-components/astro-web-components.css"
/>
<script
    type="module"
    src="https://unpkg.com/@astrouxds/astro-web-components/dist/astro-web-components/astro-web-components.esm.js"
></script>
```

You can now use any web component on your page.

```
<body>
    <rux-button>Hello World</rux-button>
</body>
```
