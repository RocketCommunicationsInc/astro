# Checkbox

A Checkbox describes a state or value that can be either "True or False.” Checkboxes are not mutually exclusive. More than one Checkbox may be checked at the same time.

For displaying invalid state, consider using a [Checkbox Group](?path=/docs/forms-checkbox-group--default-story) component.

## Guidelines

* [Astro UXDS: Checkboxes](https://www.astrouxds.com/ui-components/checkbox)
* [Astro UXDS: Form and Input Validation](https://www.astrouxds.com/ui-components/validation)

### Indeterminate

An Indeterminate state (a dash symbol rather than checked) may display when a Checkbox is used as a parent of a group of Checkboxes where at least one child is selected and at least one is not.
