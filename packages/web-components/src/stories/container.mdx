# Container

By default, Containers have a default padding of 1rem (16px) set on the body. If you'd like to change this (for example, using tables or a list inside your Container), you can use the `--body-padding` CSS custom property for convenience or the `body` CSS shadow part.

export const Default = (args) => {
    return html`
        <rux-container>
            Suscipit potenti habitant penatibus praesent quam class erat purus nisl ligula arcu accumsan risus, interdum phasellus lorem leo libero maecenas dictumst donec dis cras ante finibus.
        </rux-container>
    `
}
