### With Slotted Icon

An icon/status can be added to the left of the label by using the named [slot](https://developer.mozilla.org/en-US/docs/Web/Web_Components/Using_templates_and_slots#Adding_flexibility_with_slots) `prefix`.
For example,
```html
<rux-icon icon="apps" slot="prefix" size="auto"></rux-icon>
```
export const IconSlotted = (args) => {
    return html`
    <div style="padding: 5%;">
            <rux-accordion-item
                ?disabled="${args.disabled}"
                ?expanded="${args.expanded}">
                <rux-icon icon="apps" slot="prefix" size="auto"></rux-icon>
                <div slot="label">Title 1</div>
                <p>Content 1</p>
