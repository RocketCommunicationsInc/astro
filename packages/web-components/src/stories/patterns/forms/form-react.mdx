    ## Caveats

    ### Event Names

    Because `@astrouxds/react` components are web components at their core, their events are representative of the
    basic [HTMLElement events](https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement#events) rather than the React events. You can read about the
    difference in React's events, particularly onChange, [on the React docs page.](https://reactjs.org/docs/dom-elements.html#onchange)
