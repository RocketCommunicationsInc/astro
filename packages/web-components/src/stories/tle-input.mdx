# TLE Input

## Guidelines

The TLE Input component allows users to input and validate Two-Line Element (TLE) data, which is a standardized format for representing orbital elements of Earth-orbiting objects.

* TLE Input is fundamentally similar to a standard text area, but optimized for the specific format of TLE data
* Uses a monospace font to ensure proper alignment of TLE characters
* Validates the input against a standardized TLE format pattern
* Automatically selects all text when the user clicks in the field to simplify copy/paste operations
* Supports readonly mode for when TLEs need to be displayed but not edited

## Customization

### Horizontal Labels

You can utilize the `form-field` and `label` Shadow Parts to have full control over label placement. For example:

```css
rux-tle-input::part(form-field) {
    flex-direction: row;
}
rux-tle-input::part(label) {
    margin-right: var(--spacing-2);
}
```
