# Global Status Bar

The Global Status Bar is a full-width view across the top of an application â€” an area commonly reserved for global status, global command, and top-level navigation. The Global Status Bar often includes: Application Name, Application State, Monitoring Icons, Top Level Navigation, and an Emergency Button.

-   [Astro UXDS: Global Status Bar](https://astrouxds.com/ui-components/global-status-bar)
